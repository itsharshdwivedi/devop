ADDING THE FILE TO THE REMOTE REPOSITORY USING THE COMMAND PROMOT.

CMD COMMANDS:

STEP 01:
> git init 

STEP 02:
> git add . 

STEP 03:
> git commit -m "commit message"

STEP 04:
> git config --global user.name "your GitHub username"

STEP 05:
> git config --global user.email "your email id"

STEP 06:
> git push

STEP 07:
> git remote add <name of the file> <url>

STEP 08:
> git push <name of the file>

STEP 09:
> git commit -m "commit message"

STEP 08:
> git push --set-upstream master 



After:

STEP 01
> git add .

STEP 02:
> git commit -m "commit message"

STEP 03:
> git push 



Git Branches:

> git branch (number of branches.)
> git branch <name of new branch> (Creates a new branch.)
> git checkout <name of new branch> (Navigates to the branch specified.)


STEPS:
1. First create an empty repository.
2. > git init
3. > git add <filename.txt>
4. > git commit -m "First commit"
5. > git branch -M main
6. > git remote add origin <url>
7. > git push -u origin
8. Finally the commits will be visible on your repository.
